# 📁 04. リポジトリ構造

## 📂 ディレクトリ構成概略

```text
/
├── assets/             # 画像・アイコン等の静的アセット
├── css/                # スタイルシート (vanilla CSS)
├── js/                 # クライアントサイドJavaScript
├── wiki/               # 本ドキュメント群
├── index.html          # トップページ
├── lookbook.html       # 商品詳細・コレクション一覧
├── lookbook_backup_nav.html # バックアップファイル
├── collection.html     # 旧形式または予備のコレクションページ
├── SPEC.md             # プロジェクト仕様書
├── SEO.md              # SEO戦略
└── ... (各種スクリプト、設定ファイル)
```

## 📄 主要ファイルの解説

### ルートディレクトリ

- **index.html**: メインのランディングページ。ヒーローセクション、コンセプト、3 つのこだわり、サービスメニュー、Q&A、コンタクトフォームを含む。
- **lookbook.html**: クラシックチュチュ、ティアラ、アクセサリーの具体的な商品画像と説明を掲載する詳細ページ。
- **404.html**: エラーページ。デザインをブランドの世界観に統一している。

### assets/

- `hero_bg_optimized.jpg`: ヒーローセクションの背景画像（パフォーマンス最適化済み）。
- `miyakoya_logo.png`: メインのブランドロゴ。
- `collection_*.png`: 各コレクションカテゴリの代表画像。

### css/

- `style.css`: 全体およびトップページ用のスタイル定義。
- `lookbook.css`: コレクション詳細ページ固有のスタイル。
- `lookbook_slider.css`: スライダー等の動的要素用スタイル。

### js/

- `main.js`: スクロールアニメーション、ローディング画面、モバイルメニュー、スムーススクロールの制御。

## ⚙️ 補助スクリプト

- `optimize_image.py`: 画像の WebP 変換やリサイズを行うパイソンツール。
- `update_ogp.py`: OGP タグを自動更新するためのスクリプト。

詳細は [データモデル](./05-データモデル.md) を参照してください。
