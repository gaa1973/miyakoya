# 🚀 02. クイックスタート

## 💻 ローカル開発環境の構築

本プロジェクトはシンプルな静的サイト（HTML/CSS/JS）で構成されているため、特別なビルド手順は必要ありません。

### 1. リポジトリのクローン

プロジェクトをローカル環境に準備します。

### 2. ローカルサーバーの起動

以下のいずれかの方法で開発サーバーを起動することをお勧めします。

- **VS Code Live Server**: `index.html` を右クリックして "Open with Live Server" を選択。
- **Python**: ターミナルで `python -m http.server 8000` を実行。
- **Node.js (serve)**: `npx serve .` を実行。

### 3. ブラウザで確認

`http://localhost:8000` (または指定されたポート) を開き、最新の表示を確認します。

## 🛠️ 主な活用ツール

- **Google Chrome DevTools**: レイアウトの微調整やデバッグには必須です。
- **VS Code**: 推奨エディタ。

## 📝 修正・変更の手順

1.  ソースコードを変更する。
2.  Chrome DevTools を使用して、PC/スマホ両方の表示崩れがないか確認する。
3.  パフォーマンス（特に画像の読み込み速度）に悪影響がないか確認する。
4.  変更を保存し、[SPEC.md](../SPEC.md) のガイドラインに沿っているか再確認する。

詳細は [コントリビューションガイド](./10-コントリビューションガイド.md) を参照してください。
